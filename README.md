# AlpineReach Brand Assets
Static asset hosting for email signatures and LinkedIn graphics.
Served via assets.alpinereach-solutions.com on Vercel.

## Folder structure
- /public/email/ — Email signature images (photos, logo PNG, LinkedIn icon)
- /public/linkedin/ — LinkedIn headers and company logo
- /public/logo/ — Logo variants
- /public/people/ — Person photos

## Versioning
New filename per asset update (e.g. viktor-v2.jpg). Old URLs stay intact.
